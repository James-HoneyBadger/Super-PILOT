Overview
--------

Time Warp IDE is a comprehensive educational programming environment that supports three integrated programming languages:

* **PILOT** - A teaching language designed for computer-assisted instruction
* **BASIC** - Classic structured programming constructs
* **Logo** - Turtle graphics and procedural programming

Key Features
------------

* **Multi-Language Support**: Write programs using PILOT, BASIC, and Logo commands in a single file
* **Interactive Graphics**: Built-in turtle graphics with advanced drawing capabilities
* **Educational Focus**: Designed for teaching programming concepts
* **Extensible Architecture**: Plugin system for adding new languages and features
* **Debugging Tools**: Step-through debugging and variable watching
* **Rich Examples**: Comprehensive example programs for learning

Quick Start
-----------

1. Install dependencies:

   .. code-block:: bash

      pip install -r requirements-dev.txt

2. Run the IDE:

   .. code-block:: bash

      python Time_Warp.py

3. Load an example program from the ``examples/`` directory

4. Click "Run" to execute your program
